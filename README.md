# Aplicación de Reconocimiento de Lenguaje de Señas ASL

Sistema completo de ciencia de datos para reconocimiento de lenguaje de señas americano (ASL) usando Deep Learning y detección en tiempo real.

## Descripción del Proyecto

Este proyecto implementa un sistema end-to-end para el reconocimiento de señas del alfabeto americano (ASL) usando redes neuronales convolucionales (CNN) y Transfer Learning. Incluye análisis exploratorio exhaustivo, preprocesamiento avanzado, comparación de múltiples modelos y una aplicación de detección en tiempo real.
